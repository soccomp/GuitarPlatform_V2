# 吉他学习平台需求文档 v1.0

**整理：小霞 · 2026-04-24**

---

## 一、项目概述

**项目名称：** 吉他学习平台 v2
**定位：** 哥的专属吉他练习工具，覆盖课程学习、歌曲练习、收藏视频三大模块
**技术栈：** Vue3 + Vite（前端）、FastAPI（后端）、Whisper（音频转录）、yt-dlp（下载）

---

## 二、模块一：课程学习

### 2.1 功能流程

```
选课（视频课程列表）
    ↓
观看课程（小霞知道课程内容，可实时答疑）
    ↓
课后小霞以助教身份陪练
```

### 2.2 文件结构

每节课程为一个独立文件夹，包含：
- `video.mp4` — 课程视频
- `transcript.md` — Whisper 自动转录的文本（小霞的知识库）
- `tab.pdf` — 曲谱（如有）
- `backing.mp3` — 伴奏（如有）
- `practice.md` — 练习任务（可选，由小霞生成）

### 2.3 小霞的角色

**听课中：** 哥随时提问 → 小霞基于 transcript 回答
**听课结束：** 小霞生成练习任务 → 进入陪练模式

### 2.4 课程来源

当前：吉他新思维（持续更新）
未来：可扩展其他课程体系

## 三、模块二：歌曲练习

### 3.1 入库流程

```
哥手工整理文件 → 放到指定位置 → 平台扫描入库
```

### 3.2 支持格式

| 格式 | 说明 |
|------|------|
| PDF | 曲谱文件，支持全屏浏览 |
| GP5/GPX | GuitarPro 谱，用 alphaTab 渲染 |
| MP3 | 伴奏音频 |

### 3.3 多版本支持

同一首歌支持多个版本（如"录音室版"、"91 Live版"），在歌曲详情页展示版本列表。

### 3.4 播放功能

| 功能 | 说明 |
|------|------|
| 播放/暂停 | 标准音频控制 |
| A-B 循环 | 设置起点、终点，循环播放指定段落 |
| 速度调节 | 0.1x ~ 1.0x，按 0.1 递进 |
| 打点标记 | 在指定时间点添加标记（如"尾奏入口"），保存至数据库 |

### 3.5 曲谱浏览

- PDF / GP 全屏打开
- 全屏模式下底部显示悬浮伴奏控制条（播放/暂停/A-B/速度）
- 伴奏控制条 z-index 高于曲谱，不被遮挡

### 3.6 数据结构

```json
{
  "id": "song_001",
  "title": "灰色轨迹",
  "artist": "Beyond",
  "path": "songs/Beyond/灰色轨迹/",
  "versions": [
    {
      "name": "录音室版",
      "gp": "track.gp5",
      "pdf": "track.pdf",
      "audio": "backing.mp3"
    },
    {
      "name": "91 Live版",
      "gp": "track_live.gp5",
      "audio": "backing_live.mp3"
    }
  ],
  "markers": [
    { "version": "录音室版", "time": 12.5, "label": "尾奏入口" }
  ]
}
```

### 4.1 支持平台

- 小红书

- YouTube

### 4.2 流程

```
哥发链接 → 后端下载 → 入库 → 前端展示
```

### 4.3 命名规范

分阶段处理：
- **第一阶段：** 下载成功，可播放，保留平台原始标题

### 5.2 数据模型

统一索引文件 `data/index.json`，包含三个模块的完整数据。

### 5.3 关键技术

| 技术 | 用途 |
|------|------|
| Whisper | 本地转录，无需 API 费用 |
| alphaTab | 开源 JS 库，渲染 GP5/GPX |
| yt-dlp | 通吃三平台下载 |
| PDF.js | PDF 全屏渲染 |

## 六、技术难点（参考家驹经验）

| 难点 | 解决方案 |
|------|---------|
| GP 编码乱码 | utf-8 → latin-1 兜底（已验证有效） |
| PDF 全屏 + 伴奏叠加 | z-index + position 固定底部控制条 |
| A-B 循环 + 速度协同 | 切换速度时重置循环点，同步 currentTime |
| 多版本目录分组 | 递归扫描，按子目录识别版本 |
| 服务稳定性 | 启动脚本统一管理，减少手动操作 |
